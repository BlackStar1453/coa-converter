COA Converter Web

从零开始，两分钟完成安装

1
打开终端
在 Mac 上按 ⌘ Space 搜索 Terminal 并打开
2
克隆项目
复制下面的命令粘贴到终端，按回车
git clone https://github.com/BlackStar1453/coa-converter.git cd coa-converter
3
一键安装
自动安装所有依赖（首次可能需要输入电脑密码）
bash install.sh
4
启动服务
启动后浏览器会自动打开，终端会显示局域网地址
bash run.sh
✓
安装完成
在浏览器中上传 COA PDF，选择模板，点击转换
将下方局域网地址发给同事，他们也能直接使用
本机
http://127.0.0.1:5050
→
局域网（发给同事）
http://你的IP:5050

局域网地址在终端启动时会显示，确保设备在同一 Wi-Fi 下

⚠️ 重要提醒
网站运行在你的 Mac Air 上，只有电脑开机且服务运行时，你和同事才能访问。
电脑关机、合盖休眠、或终端关闭后，网站将无法使用。
下次使用
推荐
开机自启 — 打开电脑就能用
运行一次下面的命令，以后每次开机服务会自动启动，无需任何手动操作。
bash autostart.sh enable
关闭自启：bash autostart.sh disable  |  查看状态：bash autostart.sh status
手动启动
每次手动运行
如果不想自启，每次需要使用时打开终端运行：
cd coa-converter && bash run.sh
